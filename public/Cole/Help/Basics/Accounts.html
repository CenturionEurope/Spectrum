Cole is best used when it is with multiple people who are responsible for your website.

Cole features a comprehensive user account system that allows each user to have their own login details, settings and permissions.

It allows administrators to take control of the system and design an experience for copy writers or marketing teams etc to see and control only what is nessecary.

Cole can also be instructed to lock user permissions using the database, although it is advised that your Cole administrator applies this setting.

Setting up a new user account

It is very easy to setup a user account for use with Cole. In this section we will walk you through how to create an account, set a password and apply permissions to the account.

  1. Click your profile picture on the Cole Sidebar
  2. Under the Other Cole Users card click to view options
  3. Click Create
  4. Enter the full name of the person whom you are attributing the user account to.
  5. Provide their email address. This will be used for notifications and other Cole features such as authentication.
  6. Type a password for the user. It's best to create a strong, unique password that isn't easily guessed and isn't shared with anyone else
  7. Choose whether or not this user should use Night mode. Night mode colours parts of the interface in a darker colour pallete which is best for using Cole at night.
  8. Click Create
  9. Under the Other Cole Users card click Edit under the listing for your new user account.
  10. Click to turn on/off icons in the permissions chart. Enabling an icon will allow the user to preform this action on the relative module
  11. Click Save changes
  12. You have successfully created a new Cole User account
