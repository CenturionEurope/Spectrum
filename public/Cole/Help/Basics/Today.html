Cole welcomes you whenever you login with the Today screen. The Today screen offers a useful overview of what is happening in the system.

The today view is broken into 2 sections. The first section welcomes you with the time of day and an overview of how many notifications you have waiting for you in the system.

The second section features widgets. Widgets can be constructed if you are a competent developer, however Cole features some built in widgets to get you started

Built-in widgets

Server storage shows a basic overview of how much free space is left on your web server. This is useful if you want to upload resources to your website as, just like fuel in a car, this will depleat over time especially if you are uploading resources.

This shows the number of visitors to your site in the last 30 days as provided by Google Analytics. If Analytics has not yet been configured, You may choose to do so following the relative help document, however this is quite technical and should only be done by your Cole Administrator.

This shows the number of changes that have been made inside Cole this week. This is made up out of information being saved, created, deleted etc.

This shows how long ago Cole was updated to the newest version ensuring you get the latest features, security features and bug fixes.

This shows an overview of what is new on Cole, so you can check out exciting new features we have to offer.
