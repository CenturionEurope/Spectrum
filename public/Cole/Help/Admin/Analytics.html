Setting up Analytics is recommended only if you are comfortable editing web based files and should be preformed if you consider yourself a competent developer.

  1. Head to Google's API Site
  2. In the upper left next to the Google APIs logo, use the project selector to create a new project
  3. With the new project selected in the upper left, Click ENABLE APIS AND SERVICES
  4. Search for Analytics API and Enable it.
  5. Click Credentials on the sidebar
  6. Click the blue Create Credentials button and choose Service Account Key
  7. Under Service account, select New service account
  8. Enter a service account name
  9. Under Role select Service Accounts > Service Account Admin
  10. Under format ensure JSON is selected.
  11. A .json file is downloaded to your computer. Upload it to your server that Cole is installed on within the folder 
    /storage/app/analytics/service-account-credentials.json
  12. Open the .json file in a text editor.
  13. Look for the words 
    client_email
  14. Copy the value located near 
    client_email
    ensuring you miss out any quotes (It will look like an email address)
  15. Go to Google Analytics.
  16. Enter the admin panel
  17. Open User Management
  18. Add a new user
  19. Paste the email address you copied from the .json file into the email field.
  20. Untick Notify new users by email
  21. Tick edit, collaborate and Read & Analyse
  22. Head back to the Admin panel in Analytics and Open "View Settings"
  23. Copy the View ID number to the clipboard
  24. Add the following to the bottom of Cole's .env file 
    ANALYTICS_VIEW_ID=
  25. Paste the View ID directly after the equals sign and save changes
